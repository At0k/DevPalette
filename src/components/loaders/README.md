# Loading Components Folder

This folder contains all loading/loader component variants.

## How to Add a New Loading Component

Follow the same pattern as buttons - create a new file, export a Component object, and add it to `src/components/index.ts`.

See `src/components/buttons/README.md` for the template structure.

## Examples Included

- **Spinner** - Classic spinning circle loader
- **Skeleton Loader** - Content placeholder skeleton
- **Dots Loader** - Bouncing dots animation
